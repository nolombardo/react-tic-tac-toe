# Tic-Tac-Toe Game

A browser-based implementation of Tic-Tac-Toe built with JavaScript.

## Features

* Two-player gameplay
* Win detection logic
* Game reset functionality
* Clean, responsive UI

## Built With

* HTML
* CSS
* JavaScript
* React

## Live Site

[nolombardo.github.io/react-tic-tac-toe](https://nolombardo.github.io/react-tic-tac-toe/)

## Purpose

This project focuses on DOM manipulation, game logic implementation, and state management in React.
